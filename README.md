## 第一部分：棧的基礎概念
棧是一種 後進先出（LIFO, Last In First Out） 的資料結構。也就是說，最後壓入棧的資料會最先被取出。

棧的基本操作包括：

  Push：將數據壓入棧頂。
  
  Pop：從棧頂彈出數據。
  
  Peek/Top：查看棧頂數據，但不移除。
  
  IsEmpty：檢查棧是否為空。
  
棧的應用場景廣泛，例如：

  運算式求值與括弧匹配。
  
  深度優先搜索（DFS）。
  
  實現遞迴。
  
  流覽器前進後退功能。
